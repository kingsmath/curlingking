INTRO
컬링이란?
What is Curling?
🧊 종목 개요
빙판 위에서 스톤을 하우스(과녁) 중심에 최대한 가깝게 투척하는 전략 스포츠. 두 팀이 번갈아 스톤을 던지며 점수를 겨룬다.
🏆 정식 규격
링크 길이 42.07m, 스톤 무게 19.96kg. 팀당 4명이 각 2개씩, 한 엔드에 팀당 8개 스톤을 투척.
💡 이 게임(컬링킹)은 약식으로, 1인 vs 1인으로 진행하며 실제 규칙을 간소화했습니다. 차이점은 마지막 슬라이드에서 확인하세요.
STRUCTURE
경기 구조 — 엔드 & 선공
End System & First Player
📋 정식 경기 엔드 수
• 올림픽/세계선수권: 10엔드
• 일반 대회: 8엔드
• 팀당 한 엔드에 8개 스톤 투척
• 총 스톤 수: 16개/엔드
🎲 선공 결정 (Draw)
• 첫 엔드: 추첨으로 선공 결정
• 이후 엔드: 직전 엔드 패배 팀이 후공(해머) 획득
• 0:0 블랭크 엔드: 해머 유지
• 후공 = 해머 = 절대적으로 유리
🔨 해머(Hammer)란 한 엔드에서 마지막 스톤을 던지는 권리입니다. 마지막에 던질수록 상대 스톤을 밀어내거나 빈 공간을 노릴 수 있어 절대적으로 유리합니다.

SCORING
점수 계산 — 핵심 원리
How Scoring Works
핵심 공식:
이긴 팀 득점 = 상대 팀의 가장 가까운 스톤보다 더 중심에 가까이 있는 자기 스톤의 수
진 팀은 해당 엔드 0점 (스톤이 몇 개 있어도!)
🏅 한 엔드 최대 득점 = 팀당 스톤 수 (예: 스톤 4개면 최대 4점). 상대 스톤이 단 1개도 하우스 안에 없으면 전부 득점!
EXAMPLE A
점수 예시 ① — RED 1점
RED scores 1 point
RED 스톤이 센터에 가장 가깝고, YELLOW 최근접 스톤(기준선) 보다 안쪽에 있는 RED 스톤이 1개이므로 RED +1점. YELLOW는 0점.
EXAMPLE B
점수 예시 ② — RED 3점!
RED scores 3 points
YELLOW 최근접 스톤(Y1)의 기준선 안쪽에 RED 스톤이 R1, R2, R3 — 3개 있으므로 RED +3점. YELLOW 0점.
EXAMPLE C
점수 예시 ③ — YELLOW 2점!
YELLOW scores 2 points
RED 최근접(R1) 기준선 안쪽에 YELLOW 스톤이 Y1, Y2 — 2개. YELLOW +2점, RED 0점. R2는 하우스 밖이라 점수 계산에서 완전히 제외.
EXAMPLE D
점수 예시 ④ — 블랭크 엔드
Blank End (0:0)
하우스(원) 안에 어떤 팀의 스톤도 없으면 블랭크 엔드(Blank End) — 양 팀 모두 0점. 이 경우 해머(후공) 권한이 그대로 유지됩니다. 전략적으로 일부러 만들기도 해요!
